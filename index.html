River Worlds — Spatial DH Project

A Digital Humanities spatial visualization mapping the narrative geographies of two canonical Bengali river novels onto their real landscapes in Bangladesh.

Texts: Adwaita Mallabarman's Titas Ekti Nadir Naam (1956) & Manik Bandopadhyay's Padma Nadir Majhi (1936). Method: GIS-based narrative mapping, text-to-geography annotation.

How to publish online

  1. Save the .html file on your computer
  2. Create a free account at github.com
  3. Create a new repository → upload the file as index.html
  4. Go to Settings → Pages → Deploy from branch (main)
  5. Your site goes live at yourname.github.io/river-worlds

Alternatively: drag-and-drop the file at netlify.com — live in under a minute, free forever.

Built with: Leaflet.js (open-source mapping) + CartoDB dark tiles + Google Fonts. No server required.
